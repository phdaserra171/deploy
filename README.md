# Bradesco Livelo Página de Resgate

Esta é uma réplica offline da página de resgate de pontos Livelo do Bradesco.

## Como Usar

- Abra `index.html` em um navegador para visualizar a página.
- Para deployment online:
  - Use Surge: `surge .` (dentro desta pasta)
  - Ou GitHub Pages: Faça upload para um repositório GitHub e ative Pages.

## Arquivos Incluídos

- `index.html`: Página principal
- `assets/`: CSS, JS, imagens e fontes
- `favicon.ico`: Ícone do site

## Desenvolvimento Local

Para testar com servidor local (com salvamento de dados), use o `server.py` na pasta raiz do projeto.
